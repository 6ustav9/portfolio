Olá sou o Gustavo e trabalho como Programador na BRQ.

Apresentação

Olá sou o Gustavo, moro em São Paulo, trabalho como Programador de Sistemas de Informação na BRQ, tenho experiência em Desenvolvimento Java Web/Mobile e conhecimentos que se houver a oportunidade adoria desenvolve-los profissionalmente.

Atualmente no meu dia-a-dia de trabalho como Programador Terceiro do Banco utilizo a linguagem de Programação Java para realizar as correções em regras de negocio, funcionalidades e recursos do sistema que foi reportado no incidente, Além disso, tenho experiência em ferramentas de análise de código, incluindo decompiladores, os quais utilizo para entender o funcionamento de bibliotecas de terceiros e identificar possíveis problemas de segurança. Utilizo o Python como um auxilio nas atividades, uso ele para construir scripts de automatização de tarefas. Cheguei a utilizar o Python para limpar logs solicitados pelo cliente para facilitar na analise, visto que os logs são gigantescos.

Estou familiarizado em identificar Microserviços (Spring Framework) e PDC's (Sistemas Legados), onde os conhecimentos de PDC's (Sistemas Legados) foram aplicados para identificar onde uma variavel vinda do Mainframe esta no código Java para realização de Debug em Sala de Guerra com equipe do Mainframe, e os conhecimentos dos microserviços (spring framework) para auxiliar a equipe de microserviços se algo esta faltando implementar no código ou no microserviço ou verificar o tipo de uma variavel que esta definido no Microserviço e na aplicaçao. A IDE utilizada para desenvolvimento e montagem de ambientes é o Eclipse. Também são utilizadas Metodologias e Conceitos como: SCRUM, Lean e Agile. A utilização destas metodologias e conceitos são Kanban, Daily, Salas de Guerra e Retrospectiva.

Sou uma pessoa dedicada, empática, persistente, curiosa, pontual e proativa. Essas caracteristicas me ajudam a desenvolver minhas habilidades e consigo ver alguns progressos como: melhora na entrega, reporte, analise, triagem, performance e comunicação. No que eu conseguir ajudar meus colegas de equipe eu os ajudarei com compartilhamento de experiencias e conhecimentos
